Speech Enhancement Demo

Evaluation of MossFormerGAN Speech Enhancement in Real Acoustic Environments
GitHub Repository

Abstract

This page demonstrates speech enhancement results obtained using MossFormerGAN followed by an optimized mastering stage. The goal of this experiment is to evaluate speech enhancement performance in real acoustic environments recorded across multiple meeting rooms. A clean speech reference was played through a loudspeaker and recorded in 75 different rooms. Each recording was processed using the following pipeline:
  • MossFormerGAN speech enhancement
  • Automatic mastering optimized via Optuna
  • Objective quality evaluation using the NISQA model
The results demonstrate consistent improvements in perceptual speech quality across diverse acoustic environments.

Experiment Setup

Recording pipeline:
  • Playback of a clean speech reference
  • Recording using a room microphone
  • Speech enhancement using MossFormerGAN
  • Automatic mastering optimization
  • Evaluation via NISQA MOS prediction
Dataset size:
  • 75 room recordings
  • 1 clean reference sample

Experimental Results

Average results across all recordings show a substantial improvement in perceptual speech quality. MOS increases from 2.13 to 2.98, corresponding to an improvement of approximately 40%. The most significant improvement is observed in the NISQA noise component, indicating effective suppression of environmental noise and reverberation.
